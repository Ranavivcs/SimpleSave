# SimpleSave — פשוט לחסוך

Smart Israeli mortgage comparison & management platform. Three services: **New Mortgage**, **Refinancing (מחזור)**, and **Mortgage Insurance**. Three roles: Customer, Advisor, Admin. Hebrew-first (RTL), config-driven, modular.
